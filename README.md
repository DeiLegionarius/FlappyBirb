# FlappyBirb
Indie version of Flappy Bird made in python using Pygame.

## Changelog v1.1.0:
- improved optimization by preloading masks,
- changed starting speed, gravity and jump power,
- made difficulty 3 the starting and main difficulty,
- added pipes that move vertically (*only for difficulty 3!*),
- added global highscore (*only for difficulty 3!*),
- added sound effects and music,
- added resolution and volume settings,
- other small visual changes.
